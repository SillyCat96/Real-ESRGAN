# Real-ESRGAN — Image Upscaling Pipeline

Dockerized pipeline для апскейла изображений с помощью [Real-ESRGAN](https://github.com/xinntao/Real-ESRGAN) (×4, CPU).  
Автоматически генерирует LR-версии из эталонов, апскейлит их и считает метрики качества (PSNR / SSIM).

Мой телеграм: @SillyCat96

---

## Требования

- [Docker Desktop](https://www.docker.com/products/docker-desktop/) (Windows / Mac / Linux)
- Git

---

## Быстрый старт

```bash
git clone --config core.autocrlf=false https://github.com/SillyCat96/Real-ESRGAN.git
cd Real-ESRGAN
bash ./run.sh
```

При первом запуске Docker автоматически соберёт образ (~5–10 минут, скачает PyTorch и модель).  
Повторные запуски стартуют моментально.

---

## Структура папок

```
Real-ESRGAN/
├── benchmark/      ← положи сюда HR-эталоны (например Set14)
├── reference/      ← авторские результаты для сравнения (уже в репозитории)
├── pictures/       ← LR-версии генерируются сюда автоматически
├── results/        ← апскейленные картинки появляются здесь
├── run.sh          ← точка входа
├── make_lr.py      ← генерация LR из HR (BICUBIC ×4)
├── metrics.py      ← подсчёт PSNR / SSIM
├── Dockerfile
└── requirements.txt
```

### Апскейл одного файла

```bash
bash ./run.sh pictures/myphoto.jpg
```

### Дополнительные флаги

| Флаг | Описание |
|------|----------|
| `--scale 2` | Уменьшать LR в 2 раза вместо 4 (default: `4`) |
| `--regen` | Пересоздать LR-картинки заново (если поменял benchmark/) |
| `--no-metrics` | Пропустить подсчёт PSNR / SSIM |

Примеры:

```bash
bash ./run.sh --scale 2
bash ./run.sh --regen
bash ./run.sh --no-metrics
bash ./run.sh myphoto.jpg --no-metrics
```

После блока `results/ vs benchmark/` автоматически появляется второй блок,
если в репозитории есть папка `reference/` с заранее заготовленными результатами:

## Метрики качества

| PSNR | Оценка |
|------|--------|
| ≥ 40 dB | 🟢 Отличное |
| 30–40 dB | 🟡 Хорошее |
| 20–30 dB | 🟠 Среднее |
| < 20 dB | 🔴 Низкое |

**Примечание:** низкий PSNR на картинках с мелкой текстурой (шерсть, трава, перья) — ожидаемое поведение.  
Real-ESRGAN генерирует новые детали, которые визуально выглядят лучше, но попиксельно отличаются от оригинала.  
Для честного сравнения используй стандартные датасеты: [Set14].
